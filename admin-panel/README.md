# Админ-панель турагентства

Веб-интерфейс для администраторов турагентства.

## Технологии
- React 18
- TypeScript
- Material-UI
- React Router
- Axios
- Recharts
- Vite

## Требования
- Node.js 18+
- npm или yarn

## Установка

```bash
npm install
```

## Запуск (разработка)

```bash
npm run dev
```

Админка будет доступна на `http://localhost:3000`

Backend должен быть запущен на `http://localhost:8080`

## Сборка

```bash
npm run build
```

## Демо-доступы

- **Админ**: admin / password123
- **Пользователь**: user / password123

## Функциональность

- Авторизация по JWT
- Dashboard с метриками
- Управление турами (CRUD)
- Просмотр и обработка заявок клиентов
- Управление рейсами
- Аналитика (графики)
